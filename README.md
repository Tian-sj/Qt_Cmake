# 项目目录结构

```plaintext
Qt_CMake/
├── src/                      # 源代码目录
│   ├── gui/                  # GUI 相关代码
│   ├── RegistrationCode      # 注册码类
├── main/                     # 项目入口
├── tests/                    # 测试代码
├── scripts/                  # 构建或运行脚本
├── docs/                     # 项目文档
├── build/                    # 构建输出目录（通常添加到 .gitignore）
├── CMakeLists.txt            # 项目根目录的 CMake 配置
└── .gitignore                # Git 忽略规则
